
# My Wallet Backup Decryption Tool

## Install

```sh
npm install
```

## Run

```sh
npm run build
npm start
```

## Package

```sh
npm run build
npm run pack
```

Note for OSX users: in order to package for windows you must have `wine` installed. Install `wine` with Homebrew as follows:

```sh
brew update
brew install wine
```
